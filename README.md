# Feedback-Projekt

Kleines Webprojekt, das Texte entgegennimmt und automatisiert Feedback dazu ausgibt.  
Das Projekt wurde mit Node.js umgesetzt und besteht aus einem einfachen Frontend und einem Server, der die Verarbeitung übernimmt.

---

## Idee

Ziel ist es, zu eingegebenen Texten eine kurze Rückmeldung zu bekommen, z. B. Hinweise oder Verbesserungsvorschläge.  
Das Projekt ist als Übungs- und Lernprojekt entstanden.

---

## Technik

- Node.js  
- Express  
- HTML, CSS, JavaScript  
- Nutzung von Umgebungsvariablen (`.env`)

---

## Funktionen

- Texteingabe über eine Webseite  
- Verarbeitung der Eingabe über den Server  
- Ausgabe eines Feedback-Textes
